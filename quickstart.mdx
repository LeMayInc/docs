## Prerequisites

Before installing LeMay Commerce, confirm the following:

- An active Shopify store on any plan (Basic, Shopify, Advanced, or Plus)
- Store owner or staff account with full permissions
- A supported AI client (Claude Desktop, Cursor, or any MCP-compatible application)

## Step 1: Install the App

Navigate to the [Shopify App Store listing](https://commerce.lemay.app) and click **Add app**. Shopify will redirect you through the OAuth authorization flow. Grant all requested permissions — LeMay Commerce requires broad access to operate across all 24 commerce categories.

<Warning>
  Granting partial permissions will limit tool availability. LeMay Commerce operates best with full store access.
</Warning>

## Step 2: Select Your Plan

After installation, you will be prompted to select a subscription tier. Each tier unlocks a progressively larger set of the 129 available tools.

| Tier | Monthly | Tools Available |
|------|---------|-----------------|
| Startup | $29 | 65 |
| Small Business | $99 | 80 |
| Business | $149 | 105 |
| Corporate | $249 | 129 |
| Enterprise | $499 | 129 + Automation |
| LeMay Agency | $14,999 | 129 + Dedicated AM |

Select the tier that matches your operational requirements. You can upgrade at any time from your Shopify admin.

## Step 3: Connect Your AI Client

LeMay Commerce operates through the Model Context Protocol. Configure your MCP client to connect to your store's Commerce endpoint.

<Tabs>
  <Tab title="Claude Desktop">
    Add the following to your Claude Desktop MCP configuration (`claude_desktop_config.json`):

    ```json
    {
      "mcpServers": {
        "lemay-commerce": {
          "command": "npx",
          "args": ["-y", "lemay-commerce"],
          "env": {
            "SHOPIFY_STORE": "your-store.myshopify.com",
            "SHOPIFY_ACCESS_TOKEN": "your-access-token"
          }
        }
      }
    }
    ```
  </Tab>
  <Tab title="Cursor">
    Add to your Cursor MCP settings (`.cursor/mcp.json`):

    ```json
    {
      "mcpServers": {
        "lemay-commerce": {
          "command": "npx",
          "args": ["-y", "lemay-commerce"],
          "env": {
            "SHOPIFY_STORE": "your-store.myshopify.com",
            "SHOPIFY_ACCESS_TOKEN": "your-access-token"
          }
        }
      }
    }
    ```
  </Tab>
</Tabs>

Your access token is available in the LeMay Commerce app settings within your Shopify admin panel.

## Step 4: Execute Your First Command

With the MCP connection established, issue a natural language command to your AI client:

```
Show me today's orders
```

LeMay Commerce will route this through the intent classifier, identify the `orders_list` tool, construct the appropriate GraphQL query with today's date filter, execute it against your store's Admin API, and return the results.

Other commands to try:

```
How many units of SKU-1234 are in stock?
```

```
What was our revenue last week?
```

```
Show me customers who placed more than 3 orders this month
```

Every command follows the same pipeline: natural language in, structured Shopify data out.

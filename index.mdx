LeMay Commerce is infrastructure for modern commerce. It is not a chatbot, not an assistant, and not a dashboard overlay. It is the operating system through which Shopify stores conduct their entire operational life — orders, customers, inventory, fulfillment, analytics, marketing, and every category between — through natural language commands executed with the precision of direct API calls.

Built on the [Model Context Protocol](https://modelcontextprotocol.io), LeMay Commerce transforms plain English into exact Shopify Admin API operations across 24 commerce categories and 129 discrete tools.

## The Standard

LeMay Commerce is the first application to treat Shopify store operations as a protocol-level concern rather than a user interface concern. Every tool in the system maps directly to a Shopify Admin GraphQL mutation or query — no REST, no workarounds, no abstraction layers that obscure what is actually happening.

When you say "show me last week's orders over $500," that command passes through intent classification, parameter extraction, and GraphQL execution in a single deterministic pipeline. The result is not a summary or an interpretation. It is the data, returned exactly as Shopify's API provides it.

This is what we mean by operating system. Not a layer on top of your store. The layer your store runs on.

## v1.0.0 — March 2026

**Initial Release**

LeMay Commerce launches on the Shopify App Store with 129 tools across 24 commerce categories.

- 129 discrete commerce tools mapped to Shopify Admin GraphQL operations
- Intent classification engine for natural language command routing
- MCP server compatible with Claude Desktop, Cursor, and all MCP clients
- Six subscription tiers from Startup ($29/mo) to LeMay Agency ($14,999/mo)
- Full GDPR compliance with all mandatory Shopify webhooks
- Multi-location inventory support
- Bulk operations for Corporate tier and above
- GraphQL-exclusive architecture (no REST API calls)
